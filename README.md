# Lira Design — Landing Page

Landing page da **Lira Design**, agência de marketing 3D.

## Sobre

Página de vendas estática (HTML, CSS e JavaScript puro) com tema dark e verde neon,
formulário que envia o briefing direto para o WhatsApp e botão flutuante de contato.

## Estrutura

```
index.html   → estrutura e conteúdo
styles.css   → estilos (tema dark + verde neon, responsivo)
main.js      → animações de scroll e envio do formulário para o WhatsApp
```

## Como rodar localmente

Abra o `index.html` no navegador, ou rode um servidor local:

```bash
python -m http.server 5500
```

Acesse `http://localhost:5500`.

## Publicação (GitHub Pages)

O site é publicado automaticamente via GitHub Pages a partir da branch `main`.
